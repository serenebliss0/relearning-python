## Common Beginner Mistakes to Avoid

1. **Forgetting colons**: `if x > 5:` (not `if x > 5`)
2. **Wrong indentation**: Python uses 4 spaces (or consistent tabs)
3. **Comparing with `=` instead of `==`**: `if x == 5` (not `if x = 5`)
4. **Modifying lists while iterating**: Create a copy first
5. **Not reading error messages**: They tell you exactly what's wrong!
